 Weather App | تطبيق الطقس

نبذة عن المشروع
هذا مشروع تطبيق طقس بسيط وعملي، يخليك تعرف حالة الطقس لأي مدينة في العالم.  
المشروع معمول باستخدام HTML, CSS, JavaScript ويعتمد على Weather API لجلب بيانات الطقس الحقيقية.

المشروع هذا معمول كمشروع دراسي لمادة برمجة الإنترنت المتقدمة.


  شن يدير التطبيق؟
التطبيق يوفرلك الميزات التالية:

-  البحث عن حالة الطقس حسب اسم المدينة  
-  عرض درجة الحرارة الحالية  
-  عرض وصف الطقس (مشمس، غائم، ممطر، إلخ)  
-  توقعات الطقس لعدة أيام (Forecast)    
-  تغيير الخلفية حسب حالة الطقس  
-  عرض خريطة مرتبطة بالمدينة المختارة
